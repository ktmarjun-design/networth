Every category has three fields. Mutual Fund, SIF, NPS & Shares fetch NAV/price automatically. GIFT City can auto-fetch too if you add its fund-house NAV page URL.
How value is computed
• Mutual Fund / SIF — Name · Qty · NAV (auto, AMFI via mfapi.in) → Qty × NAV
• Shares — Name · Qty · Price (auto, NSE/BSE via relay) → Qty × Price
• NPS — Name · Qty · NAV (auto, npsnav.in) → Qty × NAV
• GIFT City — Name · Qty · NAV (USD) + optional NAV-page URL. If a URL is set, the app reads the latest USD NAV from that page via a relay (best-effort); else it uses your typed NAV. → Qty × NAV(USD) × live USD–INR.
• FD / Savings / Insurance / Others — Name · Qty · Value (₹) → Value as-is.
